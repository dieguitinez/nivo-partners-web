Accessibility Statement

Last updated: February 19, 2026  |  Standard: WCAG 2.1 Level AA

Our Commitment

Nivo Partners, LLC is committed to ensuring that our digital presence is accessible to everyone, including people with visual, auditory, motor, or cognitive disabilities. We continuously work to improve accessibility in conformance with:

  • ADA — Americans with Disabilities Act (Title III)
  • WCAG 2.1 Level AA — Web Content Accessibility Guidelines by W3C
  • Section 508 — Federal accessibility standards (reference)

Current Conformance Status

nivopartners.com is partially conformant with WCAG 2.1 Level AA. We meet most criteria and continue to address remaining gaps.

What We Do

Area Practice
Color contrast Minimum 4.5:1 ratio for normal text
Keyboard navigation All interactive elements are reachable via Tab
Alternative text All informative images include descriptive alt attributes
Heading structure Consistent H1→H2→H3 hierarchy across all pages
Forms All form fields have associated labels
Links All links use descriptive text (no "click here")
Typography Minimum 16px body text size
Responsive design Content works with up to 200% browser zoom

Known Limitations

Item Action Plan Target
Kai chatbot missing full ARIA announcement on open Add aria-live="polite" to chat container Q1 2026
CSS animations lack reduced-motion option Implement prefers-reduced-motion media query Q1 2026

Assistive Technologies

Our site has been tested for compatibility with:

  • Screen readers: NVDA (Windows), VoiceOver (macOS/iOS), TalkBack (Android)
  • Keyboard navigation: Tab, Enter, Escape, arrow keys
  • Browser zoom: Up to 200% without content loss
  • High contrast mode: Compatible with Windows and macOS high-contrast settings
  • Browsers: Chrome, Firefox, Safari, Edge (current versions)

Evaluation Approach

We evaluate the accessibility of our site through:

  1. Automated audits: Google Lighthouse (Accessibility section) — target: ≥90/100
  2. Manual review: Heading structure, keyboard navigation, and color contrast checks on each deploy
  3. Ongoing review: Any time new pages or features are added to the site

Report an Accessibility Issue

If you encounter any accessibility barrier on our site, or if you need content in an alternative format, please contact us:

Email: accessibility@nivopartners.com
Subject: [Accessibility] Brief description of the issue
Include: Page URL, issue description, and assistive technology you were using (if applicable)

We will respond within 5 business days with a solution or an action plan with a target date.

Formal Complaints

If you are not satisfied with our response, you may contact the U.S. Department of Justice's ADA Information Line at 1-800-514-0301 (voice) or 1-800-514-0383 (TTY).

Updates to This Statement

This statement was first published on February 19, 2026. It is reviewed and updated whenever significant changes are made to the site.
