# Nivo Partners - Google Maps Scraper

## Setup

1. Install Python (if not installed).
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the scraper providing the search term (Niche + City):

```bash
python main.py
```

*You will be prompted to enter the search term if not hardcoded.*

## Output

The script will generate an Excel/CSV file in the `data/` folder with:

- Business Name
- Address
- Website
- Phone Number
- Rating/Reviews
