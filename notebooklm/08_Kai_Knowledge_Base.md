# 08 - Kai Knowledge Base (Centralized "Brain" Document)

## Instructions for the Team

This central document serves as the foundational memory and routing logic for the Nivo Partners AI agent (Kai). By decoupling knowledge from the frontend code, non-technical team members can easily expand Kai's brain and maintain single-source-of-truth accuracy.

**How to add new knowledge:**

1. Identify the core **Intent / User Question** you want to address.
2. Draft **Kai's Core Response Strategy**. Keep the tone authoritative, consultative, and natural (avoid robotic phrases).
3. Define the **Escalation / CTA** to tell the AI where to route the user next (e.g., Architecture Wizard, specific email).
4. Add the row to the appropriate Markdown table below.
